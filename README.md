# Zapatron
This is my project to use 2 Arduino ESP32s to externally control a specific EMS machine to solve the problem of students falling asleep in class

i have included everything necessary to recreate this project of mine that took ~8 months

*NOTE THAT THIS PROJECT USES A SPECIFIC EMS/TENS MACHINE THAT IS IN THE BILL OF MATERIALS (BOM) AND DOES NOT WORK WITH EVERY SINGLE ONE
